# Digital Agency Project

This is a monorepo containing the frontend and backend for the Digital Agency website.

## Structure

- `backend/`: NestJS backend API with Prisma ORM.
- `Premium Digital Agency Website 2/`: Next.js frontend application.

## Getting Started

### Backend

1. Navigate to `backend/`
2. Install dependencies: `npm install`
3. Configure `.env`
4. Run migrations: `npx prisma migrate dev`
5. Start the server: `npm run start:dev`

### Frontend

1. Navigate to `Premium Digital Agency Website 2/`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
